# Brain-Tumor-Detection-CNN
Brain Tumor Detection using CNN and MRI scans

This project uses Convolutional Neural Networks (CNN) to detect brain tumors from MRI scan images. 

## Features
- Image classification using Deep Learning
- CNN model built with TensorFlow/Keras
- Predicts tumor presence from MRI scans
- Visualized accuracy and loss graphs

## Tech Stack 
- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Matplotlib

 ## Results 
- Achieved high accuracy on test data
- Successfully classifies tumor vs non-tumor images

## Project Structure Brain-Tumor-Detection-CNN/ 
│── BrainTumourTest.ipynb 
│── requirements.txt 
│── screenshots/ 

## How to Run 
1. Install dependencies:
   pip install -r requirements.txt
3. Open notebook:
   jupyter notebook BrainTumourTest.ipynb
